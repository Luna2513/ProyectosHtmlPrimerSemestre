CREANDO TABLAS HTML

una tabla es un conjunto de filas y columnas

pasos para crear una tabla

  1. etiqueta table
  2. etiqueta caption
  3. etiqueta thead
  4. etiqueta tbody
  5. etiqueta tr por cada fila
  6. etiqueta td por cada celda
  7. etiqueta tfoot para el pie de la tabla
